# Roles

**Status:** Proposed
**Created:** 2026-04-15
**Author:** Kiro AI

## Problem Statement

Ork currently has atomic skills, composites for sequential execution, and playbooks for complex orchestration. However, there's no standardized way to package and reuse complete server configurations across projects. Users must manually recreate common patterns like "secure webserver", "database server", or "monitoring agent" in each project.

Ansible solves this with **roles** - reusable, self-contained units that bundle related skills, variables, files, and templates into a single package. Ork needs similar functionality to enable:
- Sharing common server configurations across projects
- Building libraries of reusable infrastructure patterns
- Organizing complex automation into logical, maintainable units
- Distributing best practices as installable packages

## Proposal

Add **Roles** to Ork - reusable packages that bundle skills, configuration, files, and metadata into a standardized directory structure. Roles provide:
- **Standardized structure** for organizing automation code
- **Dependency management** between roles
- **Variable templating** for customization
- **File/template distribution** to remote servers
- **Metadata and documentation** for discoverability
- **Registry support** for sharing and installing roles

## Motivation

### Current Limitations

Users must manually recreate common patterns:

```go
// Manual recreation in every project
func setupWebserver(node ork.NodeInterface) {
    node.RunSkill(skills.NewAptUpdate())
    node.RunSkill(skills.NewAptUpgrade())
    node.RunSkill(skills.NewSshHarden())
    node.RunSkill(skills.NewFail2banInstall())
    node.RunSkill(skills.NewUfwInstall())
    // ... 20 more lines of setup
}
```

### Proposed Solution

Package as a reusable role:

```go
// Install role once
ork.RoleInstall("github.com/ork-roles/secure-webserver")

// Use anywhere
node.RunRole("secure-webserver")
```

## Architecture

### Role Structure

A role follows a standardized directory structure:

```
roles/
└── secure-webserver/
    ├── role.yaml           # Role metadata and configuration
    ├── main.go             # Role implementation (implements RoleInterface)
    ├── defaults.yaml       # Default variables
    ├── vars.yaml           # Role-specific variables (higher priority)
    ├── files/              # Static files to copy
    │   └── config.conf
    ├── templates/          # Go templates for dynamic files
    │   └── nginx.conf.tmpl
    ├── README.md           # Documentation
    └── tests/              # Role tests
        └── main_test.go
```

### Core Types

```go
// RoleInterface defines a reusable automation role
type RoleInterface interface {
    types.RunnableInterface  // Roles are runnables
    
    // Role-specific methods
    GetMetadata() RoleMetadata
    GetDependencies() []RoleDependency
    GetDefaults() map[string]interface{}
    GetVars() map[string]interface{}
    GetFiles() []RoleFile
    GetTemplates() []RoleTemplate
}

// RoleMetadata contains role information
type RoleMetadata struct {
    Name        string   `yaml:"name"`
    Version     string   `yaml:"version"`
    Description string   `yaml:"description"`
    Author      string   `yaml:"author"`
    License     string   `yaml:"license"`
    Tags        []string `yaml:"tags"`
    MinOrkVersion string `yaml:"min_ork_version"`
}

// RoleDependency defines a role dependency
type RoleDependency struct {
    Name    string `yaml:"name"`
    Version string `yaml:"version"`  // Semver constraint
    Source  string `yaml:"source"`   // URL or registry path
}

// RoleFile represents a static file to copy
type RoleFile struct {
    Src  string      // Source path in role
    Dest string      // Destination path on remote
    Mode os.FileMode // File permissions
    Owner string     // File owner
    Group string     // File group
}

// RoleTemplate represents a template file
type RoleTemplate struct {
    Src  string      // Template source in role
    Dest string      // Destination path on remote
    Mode os.FileMode // File permissions
    Owner string     // File owner
    Group string     // File group
}

// BaseRole provides foundation for role development
type BaseRole struct {
    *types.BaseRunnable
    metadata     RoleMetadata
    dependencies []RoleDependency
    defaults     map[string]interface{}
    vars         map[string]interface{}
    files        []RoleFile
    templates    []RoleTemplate
}
```

### Role Implementation (main.go)

```go
package main

import (
    "github.com/dracory/ork"
    "github.com/dracory/ork/skills"
    "github.com/dracory/ork/types"
)

type SecureWebserverRole struct {
    *types.BaseRole
}

func NewSecureWebserverRole() types.RoleInterface {
    role := types.NewBaseRole()
    role.SetID("secure-webserver")
    role.SetDescription("Secure web server configuration")
    
    // Load metadata from role.yaml
    role.LoadMetadata("role.yaml")
    
    return &SecureWebserverRole{BaseRole: role}
}

func (r *SecureWebserverRole) Run() types.Result {
    cfg := r.GetNodeConfig()
    
    // Access role variables (merged: defaults < vars < runtime args)
    sshPort := r.GetVar("ssh_port").(int)
    enableSSL := r.GetVar("enable_ssl").(bool)
    
    // Run dependency roles first
    for _, dep := range r.GetDependencies() {
        depRole := ork.LoadRole(dep.Name)
        depRole.SetNodeConfig(cfg)
        result := depRole.Run()
        if result.Error != nil {
            return result
        }
    }
    
    // Execute role tasks
    updateSkill := skills.NewAptUpdate()
    updateSkill.SetNodeConfig(cfg)
    updateSkill.Run()
    
    upgradeSkill := skills.NewAptUpgrade()
    upgradeSkill.SetNodeConfig(cfg)
    upgradeSkill.Run()
    
    // Change SSH port if specified
    if sshPort != 22 {
        sshSkill := skills.NewSecuritySSHChangePort().
            SetArg("port", fmt.Sprintf("%d", sshPort))
        sshSkill.SetNodeConfig(cfg)
        sshSkill.Run()
    }
    
    hardenSkill := skills.NewSshHarden()
    hardenSkill.SetNodeConfig(cfg)
    hardenSkill.Run()
    
    fail2banSkill := skills.NewFail2banInstall()
    fail2banSkill.SetNodeConfig(cfg)
    fail2banSkill.Run()
    
    ufwSkill := skills.NewUfwInstall()
    ufwSkill.SetNodeConfig(cfg)
    ufwSkill.Run()
    
    // Copy static files
    for _, file := range r.GetFiles() {
        r.CopyFile(file)
    }
    
    // Render and copy templates
    for _, tmpl := range r.GetTemplates() {
        r.RenderTemplate(tmpl, r.GetVars())
    }
    
    // Configure SSL if enabled
    if enableSSL {
        sslSkill := skills.NewNginxEnableSSL().
            SetArg("cert_path", r.GetVar("ssl_cert_path").(string)).
            SetArg("key_path", r.GetVar("ssl_key_path").(string))
        sslSkill.SetNodeConfig(cfg)
        sslSkill.Run()
    }
    
    return types.Result{
        Changed: true,
        Message: "Secure webserver role applied successfully",
    }
}
```

## API Design

### Loading and Running Roles

```go
// Load role from local directory
role := ork.LoadRole("secure-webserver")

// Run role on node
node := ork.NewNodeForHost("server.example.com")
results := node.RunRole(role)

// Or by name (if registered)
results := node.RunRoleByName("secure-webserver")

// Override variables at runtime
node.SetArg("ssh_port", "2222").
    SetArg("enable_ssl", "true")
results := node.RunRole(role)
```

### Installing Roles

```go
// Install from Git repository
err := ork.RoleInstall("github.com/ork-roles/secure-webserver")

// Install specific version
err := ork.RoleInstall("github.com/ork-roles/secure-webserver@v1.2.0")

// Install from local path
err := ork.RoleInstallLocal("./my-roles/custom-role")

// List installed roles
roles := ork.RoleList()
for _, role := range roles {
    fmt.Printf("%s v%s - %s\n", role.Name, role.Version, role.Description)
}

// Update role
err := ork.RoleUpdate("secure-webserver")

// Remove role
err := ork.RoleRemove("secure-webserver")
```

### Role Registry

```go
// Search for roles
results := ork.RoleSearch("webserver")
for _, role := range results {
    fmt.Printf("%s - %s\n", role.Name, role.Description)
}

// Get role info
info := ork.RoleInfo("secure-webserver")
fmt.Printf("Version: %s\n", info.Version)
fmt.Printf("Dependencies: %v\n", info.Dependencies)
```

### Variable Precedence

Variables are merged with the following precedence (lowest to highest):

1. Role defaults (defaults.yaml)
2. Role vars (vars.yaml)
3. Inventory group vars
4. Inventory host vars
5. Runtime args (SetArg)

```go
// Role defaults: ssh_port = 22
// Role vars: ssh_port = 2222
// Runtime override
node.SetArg("ssh_port", "3333")
results := node.RunRole(role)
// Uses ssh_port = 3333
```

    *types.BaseRole
}

func (r *NginxRole) Run() types.Result {
    cfg := r.GetNodeConfig()
    
    // Copy static configuration file
    r.CopyFile(RoleFile{
        Src:  "files/nginx.conf",
        Dest: "/etc/nginx/nginx.conf",
        Mode: 0644,
        Owner: "root",
        Group: "root",
    })
    
    // Render template with variables

    r.RenderTemplate(RoleTemplate{
        Src:  "templates/site.conf.tmpl",
        Dest: "/etc/nginx/sites-available/default",
        Mode: 0644,
    }, vars)
    
    // Restart nginx
    restartSkill := skills.NewServiceRestart().
        SetArg("service", "nginx")
    restartSkill.SetNodeConfig(cfg)
    return restartSkill.Run()
}

## Implementation Considerations

### 1. Role Loading

```go
// RoleLoader handles role discovery and loading
type RoleLoader struct {
    searchPaths []string
}

func (l *RoleLoader) Load(name string) (types.RoleInterface, error) {
    // Search in order: project, user, system
    for _, path := range l.searchPaths {
        rolePath := filepath.Join(path, name)
        if exists(rolePath) {
            return l.loadFromPath(rolePath)
        }
    }
    return nil, fmt.Errorf("role not found: %s", name)
}

func (l *RoleLoader) loadFromPath(path string) (types.RoleInterface, error) {
    // Load role.yaml
    metadata := loadMetadata(filepath.Join(path, "role.yaml"))
    
    // Load defaults and vars
    defaults := loadYAML(filepath.Join(path, "defaults.yaml"))
    vars := loadYAML(filepath.Join(path, "vars.yaml"))
    
    // Load Go implementation
    role := loadGoPlugin(filepath.Join(path, "main.go"))
    
    // Set metadata and variables
    role.SetMetadata(metadata)
    role.SetDefaults(defaults)
    role.SetVars(vars)
    
    return role, nil
}
```

### 2. Dependency Resolution

```go
// DependencyResolver handles role dependencies
type DependencyResolver struct {
    loader *RoleLoader
}

func (r *DependencyResolver) Resolve(role types.RoleInterface) ([]types.RoleInterface, error) {
    deps := role.GetDependencies()
    resolved := make([]types.RoleInterface, 0, len(deps))
    
    for _, dep := range deps {
        // Check version constraint
        if !r.satisfiesVersion(dep) {
            return nil, fmt.Errorf("version constraint not satisfied: %s %s", dep.Name, dep.Version)
        }
        
        // Load dependency
        depRole, err := r.loader.Load(dep.Name)
        if err != nil {
            return nil, err
        }
        
        // Recursively resolve dependencies
        subDeps, err := r.Resolve(depRole)
        if err != nil {
            return nil, err
        }
        
        resolved = append(resolved, subDeps...)
        resolved = append(resolved, depRole)
    }
    
    return resolved, nil
}
```

### 3. Variable Merging

```go
// VariableMerger handles variable precedence
type VariableMerger struct{}

func (m *VariableMerger) Merge(role types.RoleInterface, node ork.NodeInterface) map[string]interface{} {
    vars := make(map[string]interface{})
    
    // 1. Role defaults (lowest priority)
    merge(vars, role.GetDefaults())
    
    // 2. Role vars
    merge(vars, role.GetVars())
    
    // 3. Inventory group vars (if applicable)
    if group := node.GetGroup(); group != nil {
        merge(vars, group.GetVars())
    }
    
    // 4. Inventory host vars
    merge(vars, node.GetVars())
    
    // 5. Runtime args (highest priority)
    merge(vars, node.GetArgs())
    
    return vars
}
```

### 4. File and Template Handling

```go
// FileHandler manages file and template operations
type FileHandler struct {
    cfg config.NodeConfig
}

func (h *FileHandler) CopyFile(file RoleFile) error {
    // Read source file
    content, err := os.ReadFile(file.Src)
    if err != nil {
        return err
    }
    
    // Copy to remote
    cmd := fmt.Sprintf("cat > %s << 'EOF'\n%s\nEOF", file.Dest, content)
    _, err = ssh.Run(h.cfg, types.Command{Command: cmd})
    if err != nil {
        return err
    }
    
    // Set permissions
    chmodCmd := fmt.Sprintf("chmod %o %s", file.Mode, file.Dest)
    _, err = ssh.Run(h.cfg, types.Command{Command: chmodCmd})
    
    return err
}

func (h *FileHandler) RenderTemplate(tmpl RoleTemplate, vars map[string]interface{}) error {
    // Parse template
    t, err := template.ParseFiles(tmpl.Src)
    if err != nil {
        return err
    }
    
    // Render with variables
    var buf bytes.Buffer
    if err := t.Execute(&buf, vars); err != nil {
        return err
    }
    
    // Copy to remote
    cmd := fmt.Sprintf("cat > %s << 'EOF'\n%s\nEOF", tmpl.Dest, buf.String())
    _, err = ssh.Run(h.cfg, types.Command{Command: cmd})
    
    return err
}
```

### 5. Role Registry

```go
// RoleRegistry manages role installation and updates
type RoleRegistry struct {
    baseURL string
}

func (r *RoleRegistry) Install(source string) error {
    // Parse source (git URL, registry path, etc.)
    repo := parseSource(source)
    
    // Clone repository
    destPath := filepath.Join(getRolesPath(), repo.Name)
    if err := gitClone(repo.URL, destPath); err != nil {
        return err
    }
    
    // Load and validate role
    role, err := LoadRoleLocal(destPath)
    if err != nil {
        return err
    }
    
    // Install dependencies
    for _, dep := range role.GetDependencies() {
        if err := r.Install(dep.Source); err != nil {
            return err
        }
    }
    
    return nil
}

func (r *RoleRegistry) Search(query string) ([]RoleMetadata, error) {
    // Search registry API
    resp, err := http.Get(fmt.Sprintf("%s/search?q=%s", r.baseURL, query))
    if err != nil {
        return nil, err
    }
    defer resp.Body.Close()
    
    var results []RoleMetadata
    if err := json.NewDecoder(resp.Body).Decode(&results); err != nil {
        return nil, err
    }
    
    return results, nil
}
```
