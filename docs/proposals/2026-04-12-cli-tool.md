# Proposal: CLI Tool

**Date:** 2026-04-12  
**Status:** Rejected. Out of scope.
**Author:** System Review

> **Note:** This is a new binary component, not yet started.

## Problem Statement

Currently, Ork is a Go library that requires writing Go code to use. While this provides type safety and flexibility, it creates friction for:

- Quick ad-hoc operations
- Non-Go developers
- Shell scripting and automation
- Learning and experimentation

Ansible's success is partly due to its simple CLI: `ansible-playbook site.yml`

## Proposed Solution

Create a `ork` CLI tool that wraps the library functionality with a user-friendly command-line interface.

## CLI Design

### Basic Commands

```bash
# Run a playbook
ork run <playbook> --host <host> [options]

# List available playbooks
ork list

# Show playbook details
ork describe <playbook>

# Run with inventory
ork run <playbook> --inventory inventory.json --group webservers

# Dry-run mode
ork run <playbook> --host <host> --dry-run

# Parallel execution
ork run <playbook> --inventory inventory.json --parallel 10
```

### Configuration

```bash
# Use config file
ork run apt-upgrade --config server.yml

# Override config values
ork run apt-upgrade --config server.yml --host override.example.com

# Environment variables
export ORK_SSH_KEY=id_rsa
export ORK_SSH_USER=root
ork run ping --host server.example.com
```

### Advanced Features

```bash
# Pass arguments to playbooks
ork run user-create --host server.example.com --arg username=john

# Multiple hosts
ork run apt-upgrade --host web1.example.com,web2.example.com,web3.example.com

# Verbose output
ork run apt-upgrade --host server.example.com -v
ork run apt-upgrade --host server.example.com -vv  # More verbose

# Output formats
ork run ping --host server.example.com --output json
ork run ping --host server.example.com --output yaml
```

## Implementation

### Main Command Structure

```go
package main

import (
    "github.com/spf13/cobra"
    "github.com/dracory/ork/playbook"
    "github.com/dracory/ork/playbooks"
)

func main() {
    rootCmd := &cobra.Command{
        Use:   "ork",
        Short: "Ork - SSH-based server automation",
        Long:  "Ork is a Go-based infrastructure automation tool similar to Ansible",
    }
    
    rootCmd.AddCommand(runCmd())
    rootCmd.AddCommand(listCmd())
    rootCmd.AddCommand(describeCmd())
    rootCmd.AddCommand(versionCmd())
    
    if err := rootCmd.Execute(); err != nil {
        os.Exit(1)
    }
}
```

### Run Command

```go
func runCmd() *cobra.Command {
    var (
        host       string
        port       string
        user       string
        sshKey     string
        configFile string
        inventory  string
        group      string
        dryRun     bool
        parallel   int
        args       []string
        verbose    int
        output     string
    )
    
    cmd := &cobra.Command{
        Use:   "run <playbook>",
        Short: "Run a playbook",
        Args:  cobra.ExactArgs(1),
        RunE: func(cmd *cobra.Command, args []string) error {
            playbookName := args[0]
            
            // Load config
            cfg, err := loadConfig(configFile, host, port, user, sshKey)
            if err != nil {
                return err
            }
            
            // Parse playbook args
            cfg.Args = parseArgs(args)
            
            // Get playbook
            registry := buildRegistry()
            pb, ok := registry.PlaybookFindByID(playbookName)
            if !ok {
                return fmt.Errorf("playbook '%s' not found", playbookName)
            }
            
            // Execute
            if inventory != "" {
                return runWithInventory(pb, inventory, group, parallel, dryRun)
            }
            
            return runSingle(pb, cfg, dryRun, verbose, output)
        },
    }
    
    cmd.Flags().StringVar(&host, "host", "", "Target host")
    cmd.Flags().StringVar(&port, "port", "22", "SSH port")
    cmd.Flags().StringVar(&user, "user", "root", "SSH user")
    cmd.Flags().StringVar(&sshKey, "key", "id_rsa", "SSH key filename")
    cmd.Flags().StringVar(&configFile, "config", "", "Config file")
    cmd.Flags().StringVar(&inventory, "inventory", "", "Inventory file")
    cmd.Flags().StringVar(&group, "group", "", "Inventory group")
    cmd.Flags().BoolVar(&dryRun, "dry-run", false, "Preview changes without applying")
    cmd.Flags().IntVar(&parallel, "parallel", 5, "Max parallel executions")
    cmd.Flags().StringArrayVar(&args, "arg", []string{}, "Playbook arguments (key=value)")
    cmd.Flags().CountVarP(&verbose, "verbose", "v", "Verbose output")
    cmd.Flags().StringVar(&output, "output", "text", "Output format (text, json, yaml)")
    
    return cmd
}
```

### List Command

```go
func listCmd() *cobra.Command {
    return &cobra.Command{
        Use:   "list",
        Short: "List available playbooks",
        Run: func(cmd *cobra.Command, args []string) {
            registry := buildRegistry()
            
            fmt.Println("Available playbooks:")
            for _, pb := range registry.PlaybookList() {
                fmt.Printf("  %-20s %s\n", pb.GetID(), pb.GetDescription())
            }
        },
    }
}
```

### Describe Command

```go
func describeCmd() *cobra.Command {
    return &cobra.Command{
        Use:   "describe <playbook>",
        Short: "Show detailed playbook information",
        Args:  cobra.ExactArgs(1),
        RunE: func(cmd *cobra.Command, args []string) error {
            registry := buildRegistry()
            pb, ok := registry.PlaybookFindByID(args[0])
            if !ok {
                return fmt.Errorf("playbook '%s' not found", args[0])
            }
            
            fmt.Printf("ID: %s\n", pb.GetID())
            fmt.Printf("Description: %s\n", pb.GetDescription())
            
            // Show required args if available
            if documented, ok := pb.(DocumentedPlaybook); ok {
                fmt.Printf("\nArguments:\n")
                for _, arg := range documented.Arguments() {
                    fmt.Printf("  %s: %s\n", arg.Name, arg.Description)
                }
            }
            
            return nil
        },
    }
}
```

## Usage Examples

### Basic Usage
```bash
# Ping a server
ork run ping --host server.example.com

# Update packages
ork run apt-upgrade --host server.example.com --key production.prv

# Create user
ork run user-create --host server.example.com --arg username=john
```

### With Config File
```bash
# Create config
cat > prod-db.yml <<EOF
ssh:
  host: db3.example.com
  port: 40022
  user: root
  key: 2024_sinevia.prv
EOF

# Use config
ork run ping --config prod-db.yml
ork run apt-upgrade --config prod-db.yml
```

### With Inventory
```bash
# Create inventory
cat > inventory.json <<EOF
{
  "hosts": [
    {"name": "web1", "address": "web1.example.com", "groups": ["webservers"]},
    {"name": "web2", "address": "web2.example.com", "groups": ["webservers"]},
    {"name": "db1", "address": "db1.example.com", "groups": ["databases"]}
  ]
}
EOF

# Run on group
ork run apt-upgrade --inventory inventory.json --group webservers --parallel 2

# Run on all hosts
ork run ping --inventory inventory.json
```

### Dry-Run
```bash
# Preview changes
ork run apt-upgrade --host server.example.com --dry-run

# Output:
# Would perform the following actions:
#   [execute] apt-get: Would upgrade 23 packages
#     Command: apt-get upgrade -y
```
